# Інтерфейси для AI Ads Agent — Як підключити команду без Claude Code

> Дослідження: як дати доступ до AI Ads Agent команді (CEO, маркетолог, контент-менеджер, аналітик) через зручний інтерфейс з табличками, графіками і чатом.

---

## Головне питання: Навіщо щось крім Claude Code?

Claude Code — це термінал. Для розробника — ідеально. Для CEO чи маркетолога — незручно:

- Немає красивих табличок з сортуванням
- Немає графіків трендів
- Треба встановлювати на кожен комп'ютер
- Немає мобільної версії
- $25-100/місяць за кожного користувача

**Рішення:** Побудувати "обгортку" (інтерфейс), яка підключається до того ж Claude API + MCP, але показує результати красиво.

## TOP-4 рекомендації по сценаріях

### Сценарій 0: "Команда вже в Notion" — Notion як фронтенд

**Що це:** Notion databases замість VS Code. Команда працює в знайомому інтерфейсі — таблиці, дошки, галереї. Бот читає settings з Notion, пише звіти назад. Є офіційний Notion MCP Server.

**Для кого:** Команди які вже використовують Notion, хочуть табличний інтерфейс без програмування.

**Переваги:**
- Знайомий інтерфейс для всієї команди
- Мобільний додаток (iOS/Android)
- Real-time collaboration
- Notion MCP Server — нативна інтеграція з Claude
- 5 databases: Campaigns, Settings, Reports, Creatives, Alerts

**Як це виглядає:**
```
CEO відкриває Notion на телефоні:
  → Таблиця "Ad Campaigns" з Health Score кольоровими маркерами
  → Дошка з кампаніями по статусу (Active / Paused)
  → Календар звітів
  → Змінює Target CPL → бот автоматично оптимізує

Маркетолог:
  → Галерея креативів з превʼю
  → Додає новий креатив → бот аналізує Risk Score
  → Бачить алерти в реальному часі
```

**Детальна специфікація:** [09-notion-frontend-spec.md](./09-notion-frontend-spec.md)

### Сценарій 1: "Хочу ЗАВТРА" — OpenClaw + Telegram

**Що це:** OpenClaw — це open-source бот, який працює через месенджери (Telegram, WhatsApp, Slack, Discord). Під капотом він підключається до Claude API + MCP, тобто використовує ті ж самі 47 інструментів Facebook Ads.

**Як це виглядає для користувача:**

```
CEO в Telegram:
  "Як справи з рекламою?"

Бот відповідає:
  📊 Дашборд за вчора:
  • Bas Dent: $280 | 65 лідів | CPL $4.31 ✅
  • Smile Clinic: $150 | 22 ліди | CPL $6.82 ⚠️

  ⚠️ Smile Clinic: CPL на 36% вище плану
  Рекомендація: запустити /ads-optimizer
```

**Плюси:**
- Всі вже мають Telegram — нічого не треба встановлювати
- Працює з телефону
- Безкоштовне ПЗ (платите тільки за Claude API)
- 229,000+ зірок на GitHub — величезна спільнота
- Є готові скіли для Meta Ads (Adspirer, Ryze AI)
- MCP підтримується нативно — ваші 47 інструментів працюватимуть

**Мінуси:**
- Таблички в Telegram — це текст, не інтерактивні
- Немає графіків (тільки emoji-стилізація)
- Потрібен сервер де бот буде працювати 24/7
- Безпека: кожне повідомлення = API виклик = гроші
- Безпека: був випадок з data exfiltration через сторонні скіли

**Ціна:**
- OpenClaw: безкоштовно
- Сервер: $5-20/міс (VPS)
- Claude API: ~$10-50/міс (залежно від використання)
- **Разом: ~$15-70/міс на всю команду**

**Як підключити 4 людей:**
```
1. Розгорнути OpenClaw на VPS (DigitalOcean $6/міс)
2. Створити Telegram бота через @BotFather
3. Налаштувати openclaw.yaml:
   - Claude API ключ
   - MCP сервер meta-ads
   - Telegram канал
4. Додати бота в груповий чат або кожен пише в особисті
5. Всі 4 людини пишуть боту — він відповідає
```

**Один бот на всіх:** ТАК. Один OpenClaw інстанс → один Telegram бот → всі 4 пишуть.

### Сценарій 2: "Хочу красиво" — Streamlit

**Що це:** Python-фреймворк для створення веб-додатків з табличками, графіками і чатом. Ви отримуєте справжній веб-сайт з дашбордом.

**Як це виглядає для користувача:**

```
Відкриває браузер → ads-dashboard.yourcompany.com

┌──────────────────────────────────────────────────────┐
│  🏠 AI Ads Dashboard                    [Bas Dent ▼] │
├──────────────────────────────────────────────────────┤
│                                                      │
│  📈 Витрати за тиждень          📊 CPL тренд         │
│  ┌─────────────────────┐       ┌──────────────────┐  │
│  │     $1,850           │       │  $5 ─╲           │  │
│  │  ████████████████    │       │       ╲──── $3.8 │  │
│  │  ▲ +12% vs минулий  │       │                  │  │
│  └─────────────────────┘       └──────────────────┘  │
│                                                      │
│  📋 Кампанії                                         │
│  ┌────────────┬────────┬──────┬──────┬───────────┐   │
│  │ Кампанія   │ Spend  │ Leads│ CPL  │ Health    │   │
│  ├────────────┼────────┼──────┼──────┼───────────┤   │
│  │ Implants   │ $450   │ 112  │ $4.01│ +42 🟢    │   │
│  │ Braces     │ $320   │ 58   │ $5.52│ +8  🟡    │   │
│  │ Whitening  │ $280   │ 35   │ $8.00│ -25 🔴    │   │
│  └────────────┴────────┴──────┴──────┴───────────┘   │
│                                                      │
│  💬 Чат з агентом                                    │

│  │ Ви: Чому Whitening такий дорогий?            │    │
│  │                                              │    │
│  │ Агент: Whitening має CPL $8 при плані $5.    │    │

│  │ 2) CTR 0.6% — креатив не працює              │    │

│  │ аудиторію через /targeting-expert             │    │
│  └──────────────────────────────────────────────┘    │
│  [Введіть повідомлення...                    📤]     │
└──────────────────────────────────────────────────────┘
```

**Плюси:**
- Справжні інтерактивні таблиці (сортування, фільтри, пошук)
- Графіки (Plotly, Altair) — тренди, порівняння
- Чат-інтерфейс вбудований
- Один URL — всі заходять через браузер
- Безкоштовний хостинг (Streamlit Cloud)
- Пишеться на Python — відносно просто
- 90% компаній Fortune 50 використовують Streamlit

**Мінуси:**
- Мобільна версія працює, але не ідеальна
- Потрібен Python-розробник для створення
- Не витримує більше 50 одночасних користувачів
- Не такий гарний чат як в Chainlit

**Ціна:**
- Streamlit: безкоштовно
- Хостинг: безкоштовно (Streamlit Cloud) або $5-20/міс (VPS)
- Claude API: ~$10-50/міс
- **Разом: ~$10-70/міс на всю команду**

**Як підключити 4 людей:**
```
1. Написати Streamlit додаток (Python, 200-500 рядків)
2. Підключити Claude API для чату
3. Підключити Meta Ads API для даних
4. Задеплоїти на Streamlit Cloud (безкоштовно)
5. Дати URL всім 4 — ads.yourcompany.com
6. Опціонально: додати логін/пароль
```

**Один додаток на всіх:** ТАК. Один URL, всі 4 заходять через браузер.

## Мій рекомендований шлях

### Фаза 1: Зараз (1-2 дні)

**OpenClaw + Telegram** — швидкий старт для всієї команди.

Що робимо:
1. Орендуємо VPS ($6/міс DigitalOcean)
2. Ставимо OpenClaw
3. Підключаємо Claude API + MCP meta-ads
4. Створюємо Telegram бота
5. Додаємо всіх 4 людей
6. Готово — всі можуть запитувати бота з телефону

### Фаза 2: Через 2-4 тижні

**Streamlit дашборд** — додаємо красиві таблички і графіки.

Що робимо:
1. Пишемо Streamlit додаток (Python)
2. Підключаємо Meta Ads API
3. Підключаємо Claude API для чату
4. Деплоїмо на Streamlit Cloud (безкоштовно)
5. Маркетолог і аналітик переходять на дашборд
6. CEO продовжує використовувати Telegram

### Фаза 3: Через 1-2 місяці

**n8n автоматизація** — додаємо автопілот.

Що робимо:
1. Ставимо n8n на той же VPS
2. Налаштовуємо ранкові звіти → Telegram
3. Налаштовуємо алерти при проблемах
4. Налаштовуємо тижневі звіти → email

## Відповідь на ваше питання: "Один код чи окремі?"

**Для бота (OpenClaw, Streamlit):** ОДИН на всіх. Одна інсталяція, один бот, один дашборд — всі 4 людини використовують один і той же інтерфейс.

**Для Claude Code (якщо потрібен):** ОКРЕМИЙ для кожного. Але якщо є бот/дашборд, то Claude Code потрібен тільки розробнику для підтримки системи.

```
Було:  4 × Claude Code = $400-800/міс, кожен в терміналі
Стало: 1 × OpenClaw бот = ~$30-70/міс, кожен в Telegram/браузері
```

## Безпека OpenClaw — що потрібно знати

| Ризик | Рішення |
|-------|---------|
| Сторонні скіли можуть бути шкідливими | Використовуйте тільки свої скіли, не з маркетплейсу |
| API ключі на сервері | Зберігайте в env змінних, обмежте доступ до VPS |
| Кожне повідомлення = API виклик = гроші | Встановіть ліміт витрат на Claude API |
| Хтось може написати "видали всі кампанії" | Налаштуйте safety_rules: небезпечні дії потребують підтвердження |
